# Frontend - AI Attendance System

React frontend with Tailwind CSS and Framer Motion.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Features

- **Student Registration**: Upload 3-5 face photos to register a student
- **Mark Attendance**: Upload a group photo to automatically mark attendance
- **Dashboard**: View attendance records with present/absent status

## Pages

- `/login` - Login/Register page
- `/register` - Student face registration
- `/attendance` - Mark attendance from group photo
- `/dashboard` - View attendance dashboard
